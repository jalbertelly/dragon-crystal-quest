I'd like to use Python and the PyGame framework to create a 2D RPG 8-bit style game in the spirit of 'The Legend of Zelda', 'Final Fantasy'.

The plan to use https://www.pixellab.ai/ to generate the pixel-based in-game assets.

Please act as a game designer and software engineer to create a design and implementation plan for building this game. Please write the plan to a new .md file in the current working directory.

The player will control one of two characters, a Red Dragon named Russell or a Teal Dragon named Max.

The controls should be quite simple; standard four-way directional control, a melee attack, a special attack

The game will begin with the main character having breakfast at Dragon Castle. Dragon Castle is the location where players will return to between missions. It will have an area where players can choose which armor to use if they have more than one set. It will also have an area where the player can begin the next available mission.

To begin the game, the main character will have access to a basic set of armor and only melee attacks.

When a player begins a new mission they will find themselves in a new dungeon where they will have to battle through mobs to retrieve a colored crystal. Each mission will have a set of low level mobs, a mini boss, and the final boss which they will battle for the crystal.

For instance, the first mission might be to Dragon Volcano, where the player will traverse a series of rooms filled with Race Car Monsters to get to the mini-boss Rock Monster. Once the Rock Monster is defeated, the player battles through a second set of rooms to reach the center of the volcano where they battle the Lava Monster for the Red Crystal.

Once a crystal is retrieved it unlocks a new 'special power' for the player; for instance the red crystal might unlock fire breath.

To begin let's plan for three missions for the Red, Blue, and Green crystals.
